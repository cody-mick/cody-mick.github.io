Encapsulation

The inner workings are kept inside the object. Only the essential functionalities are exposed to the end user.

Polymorphism

Various objects can share the same method, but also have the ability to override shared methods with a more specific implementation. The same process can be used for different objects.

Inheritance

Taking the features of one object and then adding new features. We can take an object that already exists, inherit all its properties and methods. We can improve on its quality by adding new properties and methods.
