    Ch. 8 CSS3 Transforms and Transitions

    The CSS transform property lets you translate, rotate, scale, and/or skew any element on the page. We can manipulate an elements appearance using transform functions.

    Translation

    Translation functions allow you to move elements left, right, up, or down. These functions are similar to the behavior of the position: relative; when declaring top and left, moving elements up and down or left and right along the x and y axes. A translation function moves the elements without impacting the flow of the document.

    The translate(x,y) function moves an element x from the left and y from the top: 

                
              transform: translate(45px, -45px);

    This can also be used to only move an item horizontally or vertically, like so: 

                
              transform: translateX(45px);
              transform: translateY(-30px);

    Scaling

    The scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one value is provided, it will be used for both the x and y values, growing or shrinking your element or pseudo-element while maintaining the original aspect ratio. 

                
              transform: scale(1.5, 0.25);

    As with translate, you can also use the scaleX(x) or scaleY(y) functions. These functions will scale only the horizontal dimensions or only the vertical dimensions, respectively. The elements center will stay in the same place as its dimensions change.

    NOTE: To declare multiple transforms, only declare one transform followed by all of the functions separated by a space. 

                
              element {
                transform: translateX(40px) scale(1.5)
              }

    Rotation

    The rotate function rotates an element around the point of origin by a specified angle value. As with scale, by default the point of origin is the element's center. Generally, angles are declared in degrees, with positive degrees moving clockwise and negative moving counterclockwise. In addition to degrees, values can be provided in grads, radians, or turns. 

                
              transform: rotate(10deg);

    Skew

    The skew function specifies a skew along the x and y axes. X for x axis and Y for y axis. If the second parameter is omitted, the skew will only occur on the x axis. 

                
              transform: skew(15deg, 4deg);

    Changing the Origin Point

    The origin of a transformation can be changed using the transform-origin property 

                
              transform-origin: 0 0;

    Transitions

    4 Transition Properties

    • transition-property
    • transition-duration
    • transition-timing-function
    • transition-delay

    Or the shorthand "transition"

                
              transition: transform 0.2s ease-out 50ms;

    You can declare multiple transitions like this, but organize them in a comma separated list.

    Animations

    Declare animation keyframes like this:

                
              @keyframes animationName {
                /* Put animation keyframes here
              }

    Each key frame looks like its own CSS declaration block. There are 2 key words: from and to, which correspond to 0% and 100% respectively.

    Animation Properties

    Once the animation is declared, add it to your CSS rule block like this: 

                
              animation: animationName

    animation Shorthand property

                
              .verbose {
                animation-name: appearDisappear;
                animation-duration: 300ms;
                animation-timing-function: ease-in;
                animation-iteration-count: 1;
                animation-direction: alternate;
                animation-delay: 5s;
                animation-fill-mode: backwards;
                animation-play-state: running;
              }

              .concise {
                animation: 300ms ease-in alternate 5s backwards appearDisappear;
              }

    Here you can also declare multiple animations on something by separating your animations by a comma.
